## 2026-02-20 - Task 4: Java 实时计算 - 滑动窗口

### What was done:
- backend-java/src/main/java/com/ecommerce/marketing/service/TrafficFilterService.java:
  - 实现 Redis ZSET 滑动窗口算法
  - **高并发优化**:
    - 使用 Pipeline 打包 ZREMRANGEBYSCORE + ZADD + ZCARD + EXPIRE 指令，减少网络 RTT
    - 使用 UUID 作为 ZSET member，避免同一毫秒数据覆盖
    - 增加 5 分钟 TTL，防止冷数据内存泄漏
  - **时间可控性**: 引入 `java.time.Clock` 依赖注入，解决时间依赖问题
- backend-java/src/test/java/com/ecommerce/marketing/service/TrafficFilterServiceTest.java:
  - 使用 Mockito 模拟 `stringRedisTemplate.executePipelined`
  - 解决 `any()` 与重载方法的二义性问题 (`any(SessionCallback.class)`)
  - 使用 `Clock.fixed()` 模拟时间推进，避免 `Thread.sleep`
  - 验证高频点击（10次）返回 true，低频点击（1次）返回 false

### Testing:
- ✅ mvn test (用户本地执行): 7 个测试全部通过
